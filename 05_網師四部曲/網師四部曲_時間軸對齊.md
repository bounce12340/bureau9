# 網師四部曲・時間軸對齊

> 目的：把反派本傳的非編年結構，與既有正典（大石前傳、依晴篇、依晴本傳、少康史實）的時點對齊；標出必改的矛盾與開寫須守的對位規則。

---

## 一、敘事讀序 vs 世界內時序

**讀序（發文順序，教父 2 式非編年）**：
回收 → 少康 → 結網 → 刀與針

**世界內實際時序（由舊到新）**：
1. **少康**（1988–1999，史實年份）
2. **網師建線**（1999 起，結網開端）
3. **結網期案型**（1999 → 近年；連到依晴本傳的「錯字」特工屬最近端）
4. **餓狼線**：招募 → 廟口 → 那三槍歿（**約現在的六至八年前**）
5. **謝明遠線**：招募 → 政大 → 接觸依晴 → 木馬 → 撤回（**近期／現在**）
6. **回收**（**現在**，謝明遠木馬被燒之後）

> 即：讀者先看到「現在」（回收），再倒回三十年源頭（少康），再看網師編年（結網），最後深掘兩個資產的隕落（刀與針）。回收與刀與針的「謝明遠段」在世界內幾乎同期（都在現在）。

## 三、網師個人年表（推定，內部用）

- 約 1968 生（黑龍江）。
- 1988–1999：少康期間，年輕，在反情報善後裡第一次感到「那隻伸進總參的手」。
- 1999：劉連昆案清算後，受命建立對台專責線。**網師起點。**
- 1999 → 近年：結網三十年（案型／撿到餓狼／栽培謝明遠）。
- 現在：約 55–58 歲。

> present-year 為內部推算（canon 未明訂絕對年份）。以「現在≈2020 年代中」推算可使各錨自洽（網師≈55、餓狼歿≈六至八年前、少康 1992/1996/1999 落在他 24/28/31 歲）。正文不必出現西元年。

## 四、既有正典事件的相對定位

### 大石前傳（阿豪前傳）── 餓狼線的時點
- 第八集〈餓狼回來了〉→ 第九集〈那三槍〉**餓狼歿** → 第十集〈變成魔鬼的那一夜〉九局清場、**大石鞋店招牌掛上**（阿豪自此成為九局鞋店掩護）、替小雅換新布鞋。
- 第十一集〈例行臨檢〉筱愛上門 → 第十四集〈走入婚姻〉婚禮＋小虎（**現 5 歲**）尾聲。
- **推得：餓狼之死約在現在的六至八年前**（以小虎 5 歲＋阿豪婚戀期回推）。**九局在餓狼死時即接收其南部網絡。**

### 依晴篇 ── 謝明遠線的時點
- 依晴大四；入九局 → **約半年後**謝明遠於政大圖書館接觸 → 交往 → 木馬手機（數次任務情資外洩）→ 黃哥拆木馬 → 攤牌、謝明遠撤回（依晴篇 15）。
- **推得：謝明遠線在近期／現在。** 謝明遠 27 歲，MSS 台灣站，政大交換研究生掩護。

### 依晴本傳（觀察者的反擊）── 案型網的現在式
- 依晴得代號「校對」後，獵捕 MSS 偽裝特工：史密斯（五百萬／倒閉雞排店統編）、海關王主任、高鐵數據傳輸、流水席刺客等。
- **推得：現在，謝明遠之後。** 這些「錯字」特工＝網師案型網落在「現在」的節點。

## 五、關鍵修正：《回收》開場前提（已套進主檔）

**原矛盾**：細綱原寫「一週內兩條線同時報廢」。與正典衝突——餓狼六至八年前已死，謝明遠近期才撤，兩者不可能同週。

**修正後前提**：
- **觸發**：謝明遠木馬近期被燒，網師對台軟線報廢。
- **動作**：網師回頭起用餓狼當年（六至八年前）留下、他以為九局未必完全收乾淨的南部舊網，作為止血／重建的後備。
- **真相**：九局早在餓狼死時就接收了該網。梭循舊管道重啟＝走進一個放了多年的陷阱。
- **「兩份卷宗」**：一份是六七年前的舊結案（餓狼）、一份是這個月的新燒毀（木馬）。網師在現在，把畢生兩個最好的資產的隕落並排在桌上。
- **標題**：由〈兩條線同時斷〉改為**〈兩條斷線〉**（去除「同時」造成的時序矛盾）。
- 此版前提其實更悲：他伸手去抓的，是一條早已死了多年的後備線。

## 六、《刀與針》雙線時序註記（開寫須守）

- **餓狼硬線**（招募 → 廟口 → 那三槍歿）發生在約六至八年前；**謝明遠軟線**（招募 → 政大 → 接觸依晴 → 木馬 → 撤）在近期。**兩線相隔數年，非同時。**
- 交替敘事（刀／針輪流）是「主題對位」手法，不是聲稱兩線同時發生。每段開頭須給時點路標（如「那是六七年前」「這是去年」「現在」）。
- 〈兩份結案報告〉＝網師在現在（謝明遠撤之後），把新報告擺到餓狼那份舊報告旁邊，兩條線在他心裡合流；結構上呼應《回收》開場的兩份卷宗（閉環）。

## 七、對位對齊檢查表（開寫時必守）

- **餓狼之死**：那三槍、九局清場、大石鞋店招牌掛上等細節須與大石前傳一致。**網師端只能知道 MSS 那側**（失去資產＋網絡被吃），不能知道阿豪／鞋店／小雅換鞋等九局內部細節。
- **謝明遠**：接觸依晴的時點（入九局半年後）、木馬型號是他親手挑、攤牌那句「你今天有沒有吃飯，每次都是真的」須與依晴篇一致。**網師端只知任務層面**，不知兩人感情細節（除非謝明遠回報露出的「雜質」）。
- **案型網**：史密斯／海關／高鐵等依晴本傳「錯字」特工，作網師案型網落在「現在」的節點，時點對齊依晴本傳期。
- **少康雙層視角**：讀者得全貌；網師只有殘缺版。越秀大石、血癌之計、軍禮等九局內幕，**網師永遠不會知道**——這份落差是本部張力來源，須在敘事中不時錨回。
- **present-year**：正文不出現西元年（維持當代台灣模糊處理）；只有《少康》用 1992／1996／1999 等真實年份作史錨。
