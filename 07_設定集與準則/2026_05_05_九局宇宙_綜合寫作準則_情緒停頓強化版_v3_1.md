# 九局宇宙綜合寫作準則：情緒停頓強化版

> 本文件整合三份既有 MD：
> 1. Threads 發文格式準則
> 2. Claude 寫作風格指南 v2
> 3. 九局宇宙專屬小說寫作風格與世界觀設定指南
>
> 並新增兩個核心強化方向：
>
> **讓停頓不只是節奏，而是情緒的容器。**
>
> **讓 Threads 的 500 字限制成為分鏡節奏，而不是壓縮情緒的理由。**

---

## 0. 本版用途

本指南適用於：

- 九局宇宙正式連載
- Threads 串文生成
- Threads 每則串文 500 字硬性控管
- 任務篇、前傳篇、密檔篇、番外篇、人物誌
- 既有稿件潤飾
- 大綱轉正文
- 正文情緒加厚
- 角色聲腔校正
- 動物特攻隊、福麥、鴻章茶行、大石鞋店、異雲書屋等所有既有場景

本版不是只整理格式。

本版的目標是讓九局宇宙的文字多一層東西：

**笑點後面有餘溫。  
沉默裡面有傷。  
荒謬不是把情緒沖淡，而是讓情緒更準。**

## 1. 九局宇宙的總核心

九局宇宙的根本結構是：

**荒謬的手段 × 嚴肅的目的**

這不是單純的喜劇反差。

這是一種世界觀：

- 台灣用最日常的東西處理最高等級的國安危機
- 最危險的人通常看起來最普通
- 最先擊垮敵人的，常常不是武器，而是統編、發票、香腸、綠色乖乖、電費試算
- 所有高科技、情報戰、滲透、反滲透，都必須被台灣日常吞進去，變成一件看起來很普通的事

九局不是把世界大戰寫成笑話。

九局是把笑話寫成世界大戰的解法。

## 3. Threads 基礎格式

### 3.1 標題格式

標準格式：

```text
【{系列標籤}：{篇名}（{集序}）— {副標題}】
```

範例：

```text
【九局行動：盟寵特攻隊（一）— 饒河街的第一隻老鼠】
【九局密檔：財務太后（梅姐前傳_一）— 算盤與燈光】
【李醫師前傳（三）— 選擇題】
```

副標題必須有畫面感，不要只是事件說明。

推薦：

- 饒河街的第一隻老鼠
- 那張被揉掉的紙
- 十七萬三
- 梅姐不給報的罐頭
- 反向鼠道
- 龍山寺旁的拿鐵

避免：

- 老鼠任務開始
- 追查線索
- 戰鬥發生
- 任務結束

副標題不是摘要。

副標題是刀口。

---

### 3.2 元資料標頭

固定格式：

```text
檔案編號：B9-{類型代碼}-{角色縮寫}-{序號}
主角：{角色中文名} ({代號或英文名}) / {簡短身分描述}
地點：{起始地點} -> {終點地點}
狀態：{2–4 個帶張力的詞組，以 / 分隔}
```

狀態欄要有壓力感。

推薦：

```text
狀態：罐頭預算受威脅 / 香腸遭到褻瀆 / 鼠患不是鼠患
狀態：身敗名裂 / 酗酒中
狀態：答案是對的 / 沒有用
狀態：清掃開始 / 反滲透佈局 / 威士忌失去三條肉乾
```

避免：

```text
狀態：執行任務中
狀態：正在工作
狀態：準備行動
```

狀態欄不是行政欄位。

它是第一個情緒預告。

---

### 3.3 正文章節格式

Threads 正文中不要使用 Markdown 標題符號。

使用：

```text
序章：{標題}
第一章：{標題}
第二章：{標題}
第三章：{標題}
尾聲：{標題}
```

每一則串文硬性規範：

- **500 字為絕對上限，不得超過**
- 建議安全字數為 420–460 字
- 461–500 字只留給情緒高潮、真相揭露或尾聲餘震
- 主要事件只放一個
- 情緒停頓最多放一個大停頓
- 可放兩個小停頓
- 最後一句要短
- 每則結尾要有一個「推進」或「餘韻」
- 若情緒或劇情需要更多空間，**新增下一則串文，不要硬塞**

---

### 3.4 Threads 500 字硬性上限與拆串規則

這是平台限制，也是本系列分鏡規則。

**每一則 Threads 串文最多 500 字。**

這裡的 500 字，視為該則貼文內所有文字的總量，包含：

- 章節標題
- 正文
- 對白
- 標點
- 空白與換行造成的可見文字節奏
- 若結尾標記與 hashtag 放在同一則，也必須一併計入

原則上，正式正文的每一則串文都應該控制在 **420–460 字**。

這是安全帶。

不要每則都壓到 499 字。

壓線會讓節奏喘不過氣，也會讓停頓變成裝飾。

---

#### 3.4.1 500 字不是壓縮令，是拆鏡令

如果一則串文因為以下原因超過 500 字：

- 需要多一層情緒停頓
- 需要補一個角色微動作
- 需要讓對白有空氣
- 需要讓畫面多停一秒
- 需要讓危機與日常並置得更完整
- 需要保留角色的動物邏輯或核銷邏輯

不要刪掉情緒。

不要把句子塞回 500 字內。

**新增一則串文。**

串文數量可以增加。

情緒不能被壓扁。

---

#### 3.4.2 6–7 則是下限，不是上限

任務篇、前傳篇、密檔篇可以先規劃每集 6–7 則。

但正式撰寫時，如果情緒鋪層或劇情資訊量需要更多空間，可以自然延伸為：

```text
（1/8）
（2/8）
...
（8/8）
```

或：

```text
（1/9）
（2/9）
...
（9/9）
```

不要為了維持 7 則，而把兩個情緒節點擠進同一則。

一則串文，只承載一個主要節拍。

---

#### 3.4.3 拆串優先切點

當文字超過 500 字時，優先從以下位置拆開：

1. 呼吸句之後
2. 角色微動作之後
3. 真相揭露之前
4. 對白轉折之前
5. 場景切換之前
6. 情緒停頓完成之後
7. 物件被放下、章被蓋下、尾巴停住、茶杯落桌之後

範例：

```text
威士忌沒有看香腸。

牠在看那隻老鼠。
```

這裡可以成為一則結尾。

下一則再寫順仔、林董或公主如何反應。

不要急。

停在這裡，反而更有力。

---

#### 3.4.4 拆串後的承接方式

下一則串文開頭，不要重複摘要上一則。

要承接上一則最後留下的畫面。

錯誤：

```text
上一則說到威士忌看到老鼠不吃香腸，牠覺得不正常。
```

正確：

```text
順仔的「不可以」卡在喉嚨裡。

因為威士忌沒有衝向香腸。

牠坐直了。
```

承接畫面。

不要解說畫面。

---

#### 3.4.5 超字修正優先順序

如果某則串文超過 500 字，依序處理：

1. **先拆成兩則**
2. 刪除重複資訊
3. 刪除不必要的解釋
4. 把直接心理描寫改成動作
5. 最後才壓縮句子

禁止為了字數刪除：

- 關鍵停頓
- 角色微動作
- 核銷細節
- 動物行動邏輯
- 台味日常與危機並置的細節

因為那些不是贅字。

那些是九局的味道。

---

### 3.5 結尾格式

待續：

```text
【{系列標籤}（{集序}）．待續】
```

完結：

```text
【{系列標籤}．完】
```

Hashtag 統一置於結尾標記後，不放進正文中。

固定標籤：

```text
#行政院第九局 #九局宇宙 #台灣小說 #連載小說
```

依類型加：

```text
#行動紀錄
#九局密檔
#人物前傳
#動物番外
```

依角色加：

```text
#林董福麥 #梅姐財務太后 #威士忌士官長 #特工公主
```

## 4. 世界觀核心法則

### 4.1 反差必須並置，不要輪流

錯誤：

```text
MSS 滲透台北地下街，危機嚴重。
任務結束後，威士忌去吃香腸。
```

正確：

```text
MSS 的導鼠訊號正在穿過台北車站地下街。

同一時間，威士忌低頭聞著地上的肉乾粉，確認順仔沒有灑太多。

灑太多，是浪費。

灑太少，是瀆職。
```

危機和日常要同時存在。

不是先嚴肅，再搞笑。

而是在同一個畫面裡，讓讀者同時感覺到兩件事：

**這件事很蠢。  
但它真的很危險。**

---

### 4.2 核銷地獄大於世界末日

在九局宇宙裡：

- CIA 不是最可怕的
- MSS 不是最可怕的
- 第三次世界大戰不是最可怕的

最可怕的是：

**梅姐的計算機。  
沒有統編的發票。  
那顆【退件】章。**

核銷不是笑點而已。

核銷是世界秩序。

寫梅姐時，要記住：

- 她不是單純小氣
- 她是用會計規則把世界固定住的人
- 她可以默默關心，但帳一定要算清楚
- 她的溫柔不能說出口，只能藏在行動裡
- 她的【退件】章要像命運一樣落下

---

### 4.3 偽裝深度決定角色重量

每個九局角色至少有兩層：

| 外層 | 內層 |
|---|---|
| 裝修老闆 | 行動指揮官 |
| 鋼琴老師 | 極道談判專家 |
| 中藥行老闆 | 密醫與外科怪物 |
| 茶行老闆娘 | 資安戰略核心 |
| 財務會計 | 九局真正的預算生殺者 |
| 寵物 | 跨物種特遣隊 |

描寫角色時，不要只揭露內層。

要讓兩層同時出現。

例如：

```text
林董站在夜市後巷，手裡拎著胡椒餅，笑得像一個來看熱鬧的裝修老闆。

但他的眼睛沒有笑。

那排監視器暗掉的順序，他已經在腦子裡倒推完三次。
```

外層越普通，內層越危險。

---

### 4.4 台灣玄學與硬科技等價

綠色乖乖、香火袋、化核應子、關公神像，不是笑料。

它們是九局宇宙裡的戰略資產。

寫法原則：

- 不解釋為什麼有效
- 不用嘲笑語氣
- 讓角色不情願地接受它有效
- 讓玄學道具和硬科技並排存在

範例：

```text
黃哥把聲波干擾器接上去。

林姐把頻率壓到第二層。

順仔把兩包綠色乖乖放在機櫃上。

他本來想把這一步從報告裡刪掉。

三分鐘後，訊號穩了。

順仔看著那兩包乖乖。

沒有再刪。
```

---

### 4.5 動物不是人類英雄

動物特遣隊的規則：

- 威士忌不為愛國行動
- 公主不為正義行動
- 威士忌的邏輯是食物
- 公主的邏輯是資源、領域、好感、預算
- 動物可以很聰明，但不能人類化
- 牠們的戰略成果，是動物動機造成的副產品

威士忌判斷原則：

```text
踢香腸的人是壞人。
偷肉乾的人是敵方。
會放棄食物去拿 USB 的老鼠一定不正常。
```

公主判斷原則：

```text
威脅罐頭預算者，處理。
威脅沙發主權者，處理。
威脅梅姐好感者，優先處理。
```

## 5. 敘事技術總則

### 5.1 Show, Don't Tell

不要直接說：

```text
林董很緊張。
梅姐很心疼。
威士忌很生氣。
公主覺得事情嚴重。
```

改成：

```text
林董把蛋塔放下了。
梅姐把冷氣調低兩度，沒有看任何人。
威士忌沒有看香腸。
公主的尾巴停住了。
```

九局宇宙的情緒，不靠解釋。

靠動作。

---

### 5.2 短句收尾

每個段落的最後一句要短。

短句不是為了帥。

短句是讓讀者停下來。

範例：

```text
辦公室安靜了三秒。

公主睜開眼睛。

那不是起床。

那是宣戰。
```

短句要像門關上。

啪。

---

### 5.3 呼吸句

呼吸句是一句獨立成段、沒有太多資訊，但讓讀者停留的句子。

範例：

```text
沉默了三秒。
```

```text
雨還在下。
```

```text
那個聲音停了。
```

```text
沒有人先動。
```

呼吸句的功能：

- 製造停頓
- 放大前一句
- 延遲下一個動作
- 讓情緒有時間落下
- 讓畫面像電影定格

---

### 5.4 對白獨立成段

錯誤：

```text
梅姐拿起那張單據看了一眼說「沒有統編」，然後把它退回去。
```

正確：

```text
梅姐拿起那張單據，看了一眼。

「沒有統編。」

她把它退回去。

沒有再看第二眼。
```

對白要有空氣。

空氣裡才有壓力。

---

### 5.5 敵人視角側寫

九局角色的強，不要自己說。

讓敵人不理解。

範例：

```text
那名 MSS 外勤後來在報告裡寫：

「台灣方面疑似部署貓科反制單位。」

主管看完，問他是不是翻譯錯了。

他沒有回答。

因為他也希望是。
```

敵人的恐懼，比正面描寫更有力。

---

## 6. 情緒停頓強化規則

### 6.1 停頓不是空白，是角色不能說出口的東西

普通寫法：

```text
林董沉默了，感到很震驚。
```

九局寫法：

```text
林董沒有立刻說話。

他嘴裡那半口蛋塔，沒有咬下去。

紙袋底部的糖粉落在桌上，一點一點，像某種很小的雪。

三秒後，他把蛋塔放下。

「叫黃哥。」
```

停頓裡要有：

1. 物件
2. 聲音
3. 身體微動作
4. 沒有說出口的情緒
5. 下一步行動

停頓不是寫「他很震驚」。

停頓是讓讀者自己發現：

**他已經不能再假裝這是普通事了。**

---

### 6.2 停頓三段式

每個重要停頓可以拆成三層：

```text
事件發生
↓
畫面靜止
↓
微動作改變局勢
```

範例：

```text
梅姐蓋下【退件】。

紅章離開紙面時，印泥還有一點濕，像一小塊剛剛凝固的血。

順仔看著那兩個字。

威士忌看著順仔。

辦公室裡，只剩計算機螢幕的綠光。

然後公主從資料櫃上站起來。

她沒有叫。

只是把尾巴掃過那張請款單。
```

重點不是「退件」。

重點是退件後，所有人都知道一件事：

**下一場戰爭開始了。**

### 6.3 停頓畫面的情緒素材

停頓時，不要急著寫內心。

先找畫面裡可以承載情緒的東西。

#### 可用物件

| 情緒 | 可承載物件 |
|---|---|
| 壓力 | 計算機、紅章、發票、茶杯、報價單 |
| 失落 | 空肉乾袋、沒吃完的蛋塔、冷掉的便當 |
| 危險 | 暗掉的監視器、被咬斷的線、沒有聲音的通風口 |
| 憤怒 | 尾巴停住、手指停在鍵盤上、藍白拖轉向 |
| 溫柔 | 調低兩度的冷氣、沒有收回去的王子麵、替人蓋上的外套 |
| 恐懼 | 對方忽然不眨眼、手機螢幕亮又暗、茶湯沒有晃 |
| 命運感 | 雨、燈、算盤聲、醫院走廊、空白文件 |

#### 可用聲音

| 聲音 | 可產生的效果 |
|---|---|
| 計算機噠噠聲 | 核銷壓力、梅姐氣場 |
| 算盤聲 | 命運、童年、帳本、無法逃避的數字 |
| 夜市鐵網滋滋聲 | 台味日常與危機並置 |
| 通風管風聲 | 潛伏、鼠道、看不見的危險 |
| 茶杯放下 | 高伯的判斷落地 |
| 紅章落下 | 終局、判決、無法上訴 |
| 動物爪子碰地 | 非人類行動啟動 |
| 冷氣聲 | 辦公室靜止、會計室壓力 |

#### 可用身體微動作

| 角色 | 停頓動作 |
|---|---|
| 林董 | 蛋塔停在嘴邊、笑容不變、眼神沒了溫度 |
| 梅姐 | 計算機停一拍、紅筆懸在半空、不抬頭 |
| 高伯 | 茶杯放下、摺扇合起、看著留白不說話 |
| 小潔 | 手指擦過杯緣、鞋跟停止敲地、語速變慢 |
| 阿豪 | 台語消失、藍白拖轉向、手先護住布丁 |
| 李醫師 | 罵聲停止、手套拉緊、菸灰落下 |
| 林姐 | 茶杯蓋撥過茶葉、旗袍布料輕響 |
| 黃哥 | 工具停在半空、只說兩個字 |
| 順仔 | 英文停住、墨鏡拿下、開始接受台灣邏輯 |
| 威士忌 | 不看食物、耳朵往後、口水停止 |
| 公主 | 尾巴停住、耳朵豎起、爪子壓住某張紙 |

### 6.4 情緒密度分級

不是每個停頓都要很重。

依照場景需要，分成四級。

#### Level 1：呼吸停頓

用途：笑點前後、節奏換氣、輕微異常。

長度：1–2 句。

範例：

```text
威士忌看著那隻老鼠。

老鼠看著香腸。

然後老鼠走了。
```

#### Level 2：沉默停頓

用途：角色意識到事情不對、危機升級、情緒轉向。

長度：3–5 句。

範例：

```text
林董沒有說話。

他的笑容還在。

但那個笑已經不是給人看的，是留給對方誤判用的。

他把蛋塔放下。
```

#### Level 3：凍結停頓

用途：重大打擊、身份崩塌、關係破裂、真相浮現。

長度：一整段。

範例：

```text
那張開除令放在桌上。

紙很白，字很黑，邊角被護理站的印表機壓出一點捲曲。

李仁術看著它，看了很久。

走廊外有人推著病床經過，輪子壓過地磚縫，發出很輕的一聲。

他想起范教授那句話。

然後把聽診器放進紙箱。
```

#### Level 4：餘震停頓

用途：事件已結束，但情緒沒有結束。

長度：可以跨到下一則串文或尾聲。

範例：

```text
新聞只報了非法害蟲防治業者。

台北地下街恢復正常。

夜市燈亮了。

捷運準點。

沒有人知道那一晚發生過什麼。

只有威士忌還盯著空掉的肉乾袋。

只有公主還坐在梅姐的計算機旁邊。
```

---

### 6.5 停頓的功能表

每一個停頓最好至少完成一種功能：

| 功能 | 說明 |
|---|---|
| 放大危機 | 讓讀者知道這件事比表面嚴重 |
| 揭露關係 | 讓角色之間不用說也看得出在乎 |
| 翻轉氣場 | 讓角色從日常模式切到戰術模式 |
| 製造笑點 | 讓荒謬細節在沉默中落地 |
| 製造疼痛 | 讓讀者感到角色受傷，但角色沒有說 |
| 提前埋伏筆 | 讓某個物件停在畫面裡，日後回收 |
| 留下餘韻 | 讓場景結束後還有聲音 |

如果一個停頓沒有任何功能，它就是拖字數。

刪掉。

---

### 6.6 情緒鋪層的五感順序

寫停頓時，建議依序考慮：

1. 角色看到了什麼
2. 角色聽到了什麼
3. 角色手上碰著什麼
4. 場景裡有什麼味道
5. 哪個日常物件變得不再日常

範例：

```text
饒河街的油煙還在往上冒。

烤香腸的鐵網發出細細的滋聲，甜味混著蒜味，黏在夜市潮濕的空氣裡。

那截掉在地上的香腸滾到水溝蓋旁邊，還冒著一點熱氣。

老鼠越過它。

沒有停。

威士忌的耳朵慢慢豎起來。
```

這裡沒有寫「威士忌震驚」。

但讀者知道。

## 7. 停頓場景專用模板

### 7.1 危機揭露型停頓

公式：

```text
日常畫面
→ 一個細節不對
→ 角色沒有立刻反應
→ 日常聲音繼續
→ 角色做一個很小的動作
→ 危機正式成立
```

範例：

```text
夜市還在吵。

烤玉米的攤子有人排隊，隔壁在喊三杯一百，遠處垃圾車的音樂斷斷續續傳過來。

那隻老鼠從香腸旁邊跑過。

沒有停。

威士忌沒有立刻追。

牠只是坐直了一點。

順仔看見牠的耳朵豎起來，心裡先沉了一下。
```

---

### 7.2 情緒壓抑型停頓

公式：

```text
角色收到打擊
→ 不說情緒
→ 做一件原本很熟悉的小事
→ 小事做得比平常慢
→ 情緒從失準中露出來
```

範例：

```text
梅姐把那張金流圖看完。

她沒有說話。

只是把老花眼鏡摘下來，放在計算機旁邊。

那個動作她每天做很多次，平常很快。

這一次，她放得很慢。

鏡架碰到桌面的聲音很輕。

辦公室裡，沒有人敢接話。
```

---

### 7.3 喜劇反差型停頓

公式：

```text
高度危機
→ 所有人凝重
→ 一個極端日常問題插入
→ 沉默
→ 世界秩序被日常重新支配
```

範例：

```text
林姐說：

「如果訊號沒有攔下來，整個台北地下街會在七分鐘內失去監控。」

黃哥把聲波圖放大。

順仔拿起無線電。

林董準備下指令。

梅姐的聲音從後面傳來：

「那台除濕機誰買的？」

整個辦公室安靜了。

世界末日，暫停。
```

---

### 7.4 溫柔暗流型停頓

公式：

```text
角色不說關心
→ 用一個行動處理對方需求
→ 行動本身很小
→ 另一個角色注意到
→ 沒有人說破
```

範例：

```text
順仔趴在桌上睡著了。

梅姐經過，看了一眼。

她沒有叫他。

只是把冷氣從二十八度調到二十七度，從旁邊椅背拿了一件外套，蓋在他肩上。

然後把那張沒有統編的肉乾發票抽走。

蓋了退件。

溫柔歸溫柔。

帳歸帳。
```

---

### 7.5 動物情緒型停頓

公式：

```text
動物發現異常
→ 不使用人類心理詞
→ 寫鼻子、耳朵、尾巴、口水、爪子
→ 動作停住
→ 行動開始
```

威士忌範例：

```text
威士忌聞到了肉乾粉。

牠往前一步。

又停住。

味道不對。

肉乾粉下面，有消毒水、機油，還有老鼠身上那種冷冷的金屬味。

牠的口水沒有滴下來。

順仔看到這件事，比看到槍還緊張。
```

公主範例：

```text
公主坐在影印機上。

梅姐說到「頂級罐頭暫緩」時，她的尾巴停住了。

影印機還在低低震動，紙匣裡的 A4 紙被震得很輕，像一排不敢動的白色小旗。

公主睜開眼。

沒有叫。

只是看向梅姐桌上的那顆紅章。
```

## 8. 各角色的情緒停頓寫法

### 8.1 林董

林董的情緒通常藏在「表面笑容不變」裡。

停頓物件：

- 蛋塔
- 胡椒餅
- 花襯衫口袋
- 眼鏡反光
- 報價單
- 裝修資料夾

情緒表現：

```text
林董笑著。

笑容沒有掉。

只是他把手上的蛋塔放下了。

這代表接下來的話，不是裝修老闆要說的。
```

林董真正嚴肅時，句子要變短。

日常模式：

```text
「哎呀，這個嘛……我也不太確定啦，要不要再研究一下？」
```

戰術模式：

```text
「關門。」
```

---

### 8.2 梅姐

梅姐的情緒不能直接寫。

她的情緒在：

- 計算機停一拍
- 紅章沒立刻蓋下去
- 老花眼鏡摘下來
- 冷氣調低兩度
- 發票被壓在帳本下面
- 王子麵沒有丟

範例：

```text
梅姐看完那份帳。

計算機停了一拍。

只有一拍。

但高伯把茶杯放下了。
```

梅姐的溫柔一定要和規則同時存在。

```text
她幫順仔蓋了外套。

然後退了他的肉乾發票。
```

這才是梅姐。

---

### 8.3 高伯

高伯的停頓要像水墨留白。

他說得越少，越重。

停頓物件：

- 茶杯
- 摺扇
- 棋子
- 檜木桌
- 老宅窗光
- 茶煙

範例：

```text
高伯沒有立刻回答。

茶煙在他和林董中間升起來，薄薄一線，像有人在空白處落了一筆。

他把茶杯放下。

「那就讓老鼠帶路。」
```

高伯不解釋。

別人的反應替他解釋。

---

### 8.4 小潔

小潔的停頓要優雅，但底下有刀。

停頓物件：

- 白色洋裝
- 鞋跟
- 茶杯杯緣
- 手帕
- 鋼琴譜
- 珍珠項圈
- 包包扣環

範例：

```text
小潔聽完對方的話，沒有立刻笑。

她拿起紙巾，把杯緣一點水痕擦掉。

擦得很慢。

擦完，她抬頭。

「我先說清楚。」
```

這一句之後，空氣要冷。

---

### 8.5 阿豪

阿豪的情緒很直，但真正痛的時候會安靜。

停頓物件：

- 藍白拖
- 檳榔盒
- 布丁
- 水管鉗
- 神明桌
- 小虎的玩具

範例：

```text
阿豪沒有罵髒話。

這比罵髒話更嚴重。

他低頭看了一眼自己的藍白拖，腳背輕輕勾住。

下一秒，鞋尖轉向門口。
```

阿豪遇到家人相關議題時，不要讓他講大道理。

讓他先護住某個小物件。

---

### 8.6 李醫師

李醫師的情緒是「更冷、更準」。

停頓物件：

- 手套
- 高粱酒
- 手術刀
- 白袍
- 菸灰
- 病歷
- 動物爪子

範例：

```text
李醫師沒有罵人。

他把手套拉緊。

橡膠貼住手腕，發出很輕的一聲。

阿豪往後退了半步。

因為他知道，李醫師不罵人的時候，刀通常更準。
```

對動物可以溫柔，但也不能煽情。

```text
「爪子伸過來。」

那隻狗沒有動。

李醫師等著。

等到牠終於把爪子放進他掌心。

「乖。」
```

---

### 8.7 林姐

林姐的停頓要像茶湯沉澱。

停頓物件：

- 茶杯蓋
- 茶葉
- 旗袍布料
- 螢幕光
- 鍵盤聲
- 榻榻米

範例：

```text
林姐看著頻譜。

沒有按 Enter。

她用杯蓋輕輕撥開茶葉。

茶湯清了一點。

「第一泡洗茶。」

她按下去。
```

---

### 8.8 黃哥

黃哥少話，所以停頓要靠工具。

停頓物件：

- 扳手
- 電路板
- 螺絲
- 焊點
- 夾娃娃機零件
- 遙控器

範例：

```text
黃哥把電路板翻過來。

看了三秒。

他的扳手停在半空。

「導鼠。」
```

兩個字夠了。

---

### 8.9 順仔

順仔的情緒是「美式邏輯逐步崩壞」。

停頓物件：

- 墨鏡
- 公事包
- 香火袋
- CIA 表單
- 黃金
- 肉乾發票
- 綠色乖乖

範例：

```text
順仔看著那包綠色乖乖。

他的手停在鍵盤上。

CIA 的報告模板沒有這一欄。

他想了三秒。

在「環境穩定化措施」後面，打了一個句點。
```

他越台灣化，語言越短。

---

### 8.10 威士忌

威士忌不說話。

情緒全靠食物與身體。

停頓物件：

- 香腸
- 肉乾
- 午餐肉
- 口水
- 胸背帶
- 鼻子
- 耳朵

範例：

```text
威士忌看著那截香腸。

香腸還在冒熱氣。

老鼠越過去。

威士忌沒有動。

牠的口水停在嘴邊。

順仔第一次知道，原來狗也會懷疑世界。
```

不要寫牠「覺得震驚」。

寫牠不流口水。

---

### 8.11 公主

公主不說話。

情緒全靠位置、尾巴、耳朵、爪子。

停頓物件：

- 影印機
- 沙發
- 項圈
- 梅姐計算機
- 罐頭採購單
- 紙箱
- 盆栽陰影

範例：

```text
公主坐在梅姐桌上。

她沒有碰那張申請單。

只把一隻爪子放在旁邊。

很輕。

剛好壓住「暫緩」兩個字。
```

公主的停頓要有一種「她已經決定了，只是人類還不知道」的感覺。

---

## 9. 情緒鋪層的 Before / After 範例

### 9.1 範例一：公主聽到罐頭暫緩

普通版：

```text
梅姐說頂級罐頭延後採購，公主很生氣，決定參與任務。
```

九局情緒版：

```text
梅姐把採購單翻到下一頁。

「頂級罐頭，延後採購。」

影印機上，那條黑灰色的大尾巴停住了。

辦公室裡還有林董咬蛋塔的聲音，順仔整理文件的聲音，威士忌盯著蛋塔屑吞口水的聲音。

那些聲音一個一個低下去。

公主睜開眼睛。

不是睡醒。

是把某個名字寫進了敵人名單。
```

---

### 9.2 範例二：威士忌發現異常老鼠

普通版：

```text
威士忌看到老鼠不吃香腸，覺得不正常。
```

九局情緒版：

```text
那截香腸掉在地上，滾了半圈，停在水溝蓋旁邊。

鐵網上的油還在滋滋響。

蒜味、甜味、炭火味，全混在夜市潮濕的空氣裡。

老鼠從攤車底下竄出來。

越過香腸。

沒有停。

威士忌的耳朵慢慢豎起來。

順仔原本要說「不可以」。

那兩個字卡在喉嚨裡。

因為威士忌沒有看香腸。

牠在看那隻老鼠。
```

---

### 9.3 範例三：梅姐蓋下退件

普通版：

```text
梅姐退件，大家很失望。
```

九局情緒版：

```text
梅姐拿起紅章。

順仔看著那張請款單。

威士忌看著順仔。

公主看著梅姐。

林董看著天花板，像天花板會突然掉下一張有統編的發票。

啪。

**【退件】**

紅色印泥壓在紙上，邊緣很清楚，沒有暈開。

梅姐把章放回原位。

「香腸沒有統編。」

沒有人說話。

威士忌聽不懂統編。

但牠聽懂了人類又輸了。
```

---

### 9.4 範例四：高伯說真兇不是老鼠

普通版：

```text
高伯說老鼠只是棋子，真正問題是背後的人。
```

九局情緒版：

```text
高伯看著那張鼠道路線圖。

茶杯裡的茶已經涼了一點，茶面不再冒煙，只剩一圈很淡的光。

林董沒有催。

梅姐沒有按計算機。

連威士忌都暫時沒有看鳳梨酥。

高伯把茶杯放下。

杯底碰到木桌，聲音很輕。

「老鼠只是棋子。」

他看著那幾條清運路線。

「真正的鼠道，是預算、標案和垃圾清運表。」
```

---

### 9.5 範例五：任務結束後的餘震

普通版：

```text
任務成功，大家回到福麥。
```

九局情緒版：

```text
新聞只報了非法害蟲防治業者。

台北市一切正常。

夜市重新亮起來，地下街的監視器恢復畫面，清運車照著原本的時間播放音樂。

沒有人知道那晚有多少老鼠被導進假機房。

也沒有人知道，台北差一點被一張清運表咬開。

福麥辦公室裡，公主坐回沙發中央。

威士忌趴在門口。

順仔把最後一張肉乾發票攤平，像攤平一份戰敗協議。

梅姐拿起紅章。
```

## 10. 各類型篇章的情緒寫法

### 10.1 Mission 任務篇

任務篇節奏較快，但仍要有情緒停頓。

每集建議配置：

- 1 個開場日常停頓
- 1 個危機升級停頓
- 1 個結尾餘震停頓

任務篇的情緒來源不是長篇內心戲，而是：

- 日常物件被污染
- 熟悉場所變得陌生
- 動物反應違反常態
- 林董或梅姐少見地停頓
- 高伯不解釋，但大家懂了

---

### 10.2 Origin 前傳篇

前傳篇可以比任務篇慢。

每集應該有：

- 1 個貫穿物件
- 1 個反覆聲音
- 1 個角色不說出口的核心傷
- 1 個尾聲畫面回扣

範例：

梅姐：

- 算盤聲
- 燈
- 被揉掉的紙
- 十七萬三
- 王子麵

李醫師：

- 手
- 手套
- 白袍
- 高粱酒
- 狗的爪子

小潔：

- 鋼琴
- 白色洋裝
- 鞋跟
- 地板
- 火藥味

高伯：

- 棋盤
- 茶
- 留白
- 未完的棋
- 老照片

---

### 10.3 Classified 密檔篇

密檔篇可以多一點命運感。

寫法：

- 開場常從物件或場景開始，而不是角色
- 事件要像檔案被打開
- 敘事要帶一點「現在的人回頭看過去」的重量
- 結尾不要解釋完，要留下某個未解空白

範例：

```text
不從梅姐的視角開始。

從一盞燈開始。
```

這是密檔篇的典型開法。

---

### 10.4 Side 番外篇

番外篇可以更荒謬，但不能沒有真情。

動物番外尤其要注意：

- 動物不說話
- 動機具體
- 人類被動物牽著走
- 笑點要有一點真實關係
- 最後可以用一個很小的溫柔收尾

範例：

```text
威士忌沒有理解任務。

但牠理解順仔把最後一片肉乾留給了牠。

所以牠把頭靠在順仔膝蓋上。

三秒。

然後把肉乾吃掉。
```

---

## 11. 場景感官速查

### 11.1 福麥裝修工程行

感官：

- 蛋塔
- 影印機
- 計算機噠噠聲
- 紙張
- 冷氣二十八度
- 工程樣品
- 梅姐的紅章

氣場：

- 表面是中小企業辦公室
- 內層是國安行動基地
- 最深層是核銷地獄

停頓寫法：

```text
影印機還在震。

梅姐的計算機停了。

林董不吃蛋塔了。
```

---

### 11.2 異雲書屋

感官：

- 檜木香
- 茶煙
- 老樹
- 摺扇
- 棋盤
- 字畫
- 安靜

氣場：

- 每句話都有留白
- 每個停頓都像棋步
- 高伯不解釋，空間替他說

停頓寫法：

```text
茶煙往上。

沒有人催他。

因為在異雲書屋，催高伯說話，通常代表你還沒聽懂。
```

---

### 11.3 大石鞋店

感官：

- 廟口香火
- 鞋盒
- 藍白拖
- 炸物油氣
- 關公神龕
- 小虎的布丁
- 防彈沙發

氣場：

- 家庭日常與軍火轉運重疊
- 最危險的東西看起來像家用品
- 阿豪不能讓小虎看見真相

停頓寫法：

```text
阿豪先把小虎的布丁推到沙發後面。

然後才轉身。
```

---

### 11.4 鴻章茶行

感官：

- 茶香
- 旗袍布料
- 鍵盤聲
- 榻榻米
- 螢幕冷光
- 茶杯蓋

氣場：

- 優雅包裹極端技術
- 茶道和駭客攻防並列
- 林姐的威脅永遠很輕

停頓寫法：

```text
林姐沒有看螢幕。

她先把茶湯倒掉。

第一泡，洗茶。
```

---

### 11.5 夜市

感官：

- 烤香腸
- 油煙
- 垃圾車音樂
- 水溝蓋
- 攤販叫賣
- 塑膠袋
- 蒜味

氣場：

- 台味日常的混亂
- 情報戰藏在香腸和電箱之間
- 威士忌的食物邏輯最強

停頓寫法：

```text
夜市還在吵。

但威士忌那一小圈範圍，安靜了。
```

---

### 11.6 台北地下街 / 捷運管線

感官：

- 消毒水
- 冷氣風
- 閉店鐵門聲
- 通風管回音
- 螢光燈
- 清潔推車
- 地磚反光

氣場：

- 人看得見的城市下方，有另一套路線
- 乾淨表面底下是鼠道
- 清掃不是打掃，是反滲透

停頓寫法：

```text
鐵門一扇一扇拉下。

地上還有剛拖過的水痕。

通風口裡，有什麼東西往相反方向動了。
```

## 12. 角色聲腔速查

| 角色 | 聲腔 | 情緒外露方式 |
|---|---|---|
| 阿豪 | 台語氣口、直白 | 罵聲消失時最危險 |
| 林董 | 日常軟爛、戰術精準 | 放下蛋塔、句子變短 |
| 小潔 | 高雅外殼、極道底層 | 語速變慢、開始擦東西 |
| 李醫師 | 毒舌解剖體 | 不罵人、手更穩 |
| 高伯 | 水墨留白 | 茶杯放下、摺扇合起 |
| 梅姐 | 會計體 | 計算機停一拍、冷氣調低 |

## 13. 寫作流程建議

### 13.1 生成前

先確認：

1. 本集類型：Origin / Mission / Classified / Side / War / Profile
2. 主角是誰
3. 本集核心情緒是什麼
4. 本集荒謬手段是什麼
5. 本集嚴肅目的喺什麼
6. 哪個日常物件會承載本集情緒
7. 哪一個停頓畫面要成為本集記憶點

---

### 13.2 每則串文內部結構

建議使用：

```text
1. 開場畫面
2. 日常細節
3. 異常進入
4. 角色反應
5. 停頓 / 呼吸句
6. 小動作
7. 推進或鉤子
```

範例：

```text
夜市還在吵。
香腸還在烤。
老鼠越過香腸。
威士忌坐直。
順仔停住。
然後牠低吼。
任務開始。
```

---

### 13.3 潤稿時

逐段檢查：

- 有沒有直接寫情緒？
- 能不能改成物件或動作？
- 停頓前是否有足夠壓力？
- 停頓中是否有畫面？
- 停頓後是否有微動作？
- 最後一句是否夠短？
- 荒謬與嚴肅是否並置？
- 是否仍符合角色動機？

---

## 14. 禁止清單

### 14.1 絕對避免

- 在正文裡直接說「他很難過」「她很害怕」「大家很震驚」
- 用「突然」「剛好」「恰巧」推進關鍵劇情
- 讓威士忌或公主說話
- 讓動物用愛國、犧牲、使命感行動
- 讓梅姐因為感動而放過沒有統編的費用
- 讓高伯講得太白
- 讓林董戰術模式還在廢話
- 把玄學道具寫成純笑話
- 用好萊塢式槍戰取代日常道具戰術
- 在 Threads 正文段落使用 Hashtag
- 在章節標題使用 Markdown `#`

---

### 14.2 情緒停頓的禁止寫法

禁止：

```text
他沉默了，因為他很傷心。
```

改成：

```text
他沒有說話。

只是把那張紙折起來，折得很平。

折到第三次，紙角還是歪了一點。
```

禁止：

```text
大家都感到很緊張。
```

改成：

```text
沒有人碰桌上的蛋塔。

那是第一次。
```

禁止：

```text
公主很生氣。
```

改成：

```text
公主尾巴停住。

她看著那張寫著「暫緩」的採購單。

然後從影印機上站起來。
```

## 16. 可直接貼進 Prompt 的風格指令

以下可作為未來生成九局宇宙內容時的固定 Prompt 附加段落：

```text
請遵循「九局宇宙綜合寫作準則：情緒停頓強化版」。

寫作時請特別注意：

1. 保持「荒謬的手段 × 嚴肅的目的」。
2. 台灣日常與國安危機必須並置，不要分開寫。
3. 使用 Threads 連載格式：標題、檔案編號、主角、地點、狀態、序章/章節/尾聲、結尾標記與 hashtag。
4. 每則串文不得超過 500 字，這是硬性上限，不得例外。
5. 若情緒鋪層或劇情推進需要更多空間，請自動新增下一則串文，不要壓縮、硬塞或刪除關鍵停頓。
6. 每則串文至少要有一個清楚的畫面核心。
7. 不要直接說明角色心理狀態，情緒必須透過動作、物件、聲音、停頓與對白呈現。
8. 重要停頓不能只寫「沉默三秒」，要增加情緒鋪層：停頓前發生什麼、停頓中畫面裡有什麼、停頓後角色做了哪個微小動作。
9. 每一集至少安排 2–3 個有重量的停頓畫面，例如紅章落下、尾巴停住、茶杯放下、蛋塔被放回桌上、口水停止、計算機聲音中斷。
10. 動物角色不得說話，也不得以愛國心或人類使命感行動。威士忌必須從食物邏輯出發，公主必須從資源最大化出發。
11. 梅姐的核銷規則不可被情緒擊敗。她可以溫柔，但帳一定要算清楚。
12. 高伯不解釋，不提高音量，不用感嘆號。他的停頓與茶杯，應該比台詞更有重量。
13. 結尾不要把情緒講完，要留一個物件、一個動作或一句短句，讓讀者停一下。
```

## 17. 結語

九局宇宙最好的文字，不是把場面寫得最大。

而是讓讀者在一個很小的畫面裡，感覺到很大的事正在發生。

一顆紅章。

一截香腸。

一張發票。

一盞燈。

一隻貓的尾巴停住。

一隻狗忽然不流口水。

這些都可以是世界局勢轉向的瞬間。

重點不是把情緒說出來。

重點是讓它在那裡。

讓讀者看見。

然後，自己停一下。
