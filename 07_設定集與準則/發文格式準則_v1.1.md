# 行政院第九局 — Threads 發文格式準則
**版本**：v1.1
**適用範圍**：所有「九局宇宙」系列內容於 Threads 平台之發布
**最後更新**：2026-06-15

> **v1.1 更新摘要**：新增「反派本傳」發文類型（`Enemy`）與「網師」系列標籤、`Net`／`Suo` 角色縮寫、反派本傳專屬 Hashtag、網師與梭的語言風格、以及「真實案件改編原則」（§十）。其餘沿用 v1.0。

## 一、發文類型分類

| 類型代碼 | 說明 | 範例 |
|----------|------|------|
| `Origin` | 人物前傳／起源篇 | 李醫師前傳、阿豪前傳 |
| `Mission` | 行動任務篇 | 吉隆坡首戰、及第計畫 |
| `Classified` | 九局密檔（組織歷史、內部事件） | 家裡有老鼠、從重水到濁水溪 |
| `Side` | 番外篇（動物、日常、特休） | 威士忌前傳、香奈兒項圈失竊事件 |
| `War` | 世界大戰篇（Unleash Chiang 主線） | 第1至10集 |
| `Profile` | 人物誌（角色深度介紹，非劇情敘事） | 各角色的角色設計揭露 |
| `Enemy` | **反派本傳（敵方視角的連載）** | **網師四部曲** |

> 備註：`Enemy` 用於從對手（敵方情報單位）視角展開的連載。若想凸顯「鏡像對手」而非單純敵我，內部亦可採 `Rival`，但對外標籤統一沿用本表。

## 二、標題格式

### 2.1 標準格式

```
【{系列標籤}：{篇名}（{集序}）— {副標題}】
```

**欄位說明：**

- **系列標籤**：使用固定標籤組，標識人物或篇章系列（見下方標籤清單）
- **篇名**：該人物或行動的主線名稱
- **集序**：以中文數字標示，如「一」、「上」、「第一集」，跨集時保持一致
- **副標題**：單集的核心意象或轉折點，以具文學張力的詞句為主

**標準系列標籤清單：**

| 標籤 | 適用範圍 |
|------|----------|
| `九局人物誌` | 人物誌類型（Profile） |
| `九局密檔` | 組織歷史與內部事件（Classified） |
| `九局行動` | 任務篇（Mission） |
| `李醫師前傳` | 李仁術起源系列 |
| `大石前傳` | 阿豪起源系列 |
| `福麥前傳` | 林董起源系列 |
| `威士忌前傳` | 威士忌起源系列 |
| `特工公主篇` | 公主相關系列 |
| `CIA特勤篇` | 順仔台灣行動系列 |
| `鴻章茶行篇` | 林姐、黃哥相關系列 |
| `Unleash Chiang` | 世界大戰主線（第1至10集） |
| `特休假後篇` | 世界大戰後番外系列 |
| `網師` | **反派本傳系列（網師四部曲）** |

**範例：**
```
【九局人物誌：李醫師前傳（一）— 金色手術刀的斷裂】
【九局行動：吉隆坡的首戰（上）— 三億元的遙控器】
【Unleash Chiang（第3集）— 大石鞋店的神主牌】
【網師・回收（一）— 兩條斷線】
【網師・少康（六）— 越秀山的大石】
```

> 反派本傳採 `【網師・{部名}（{集序}）— {副標題}】`，部名為《回收》《少康》《結網》《刀與針》。

## 三、元資料標頭（Metadata Block）

緊接在標題之後，以獨立段落呈現，使用固定欄位格式：

```
檔案編號：{代碼}
主角：{角色中文名} ({代號或英文名}) / {簡短身分描述}
地點：{起始地點} -> {終點地點（若有移動）}
狀態：{角色當下狀態，2–4個關鍵詞，以 / 分隔}
```

### 3.1 檔案編號規則

格式：`B9-{類型代碼}-{角色縮寫}-{序號}`

| 欄位 | 說明 |
|------|------|
| `B9` | 固定前綴，代表 Bureau 9 |
| 類型代碼 | `Origin`、`Mission`、`Classified`、`Side`、`War`、`Profile`、`Enemy` |
| 角色縮寫 | 使用主角的英文縮寫：`Li`（李醫師）、`Hao`（阿豪）、`Lin`（林董）、`Jie`（小潔）、`Guo`（高伯）、`Mei`（梅姐）、`Huang`（黃哥）、`Sis`（林姐）、`Andy`（順仔）、`Whiskey`（威士忌）、`Princess`（公主）、`Yi`（依晴）、`Net`（網師）、`Suo`（梭／顧承）、`Team`（多人行動） |
| 序號 | 兩位數流水號，從 `01` 開始 |

**範例：**
```
B9-Origin-Li-01        → 李醫師前傳第一集
B9-Mission-Team-03     → 多人行動任務第三集
B9-War-Team-07         → 世界大戰第七集
B9-Enemy-Net-01        → 網師四部曲・回收第一集（網師主控）
B9-Enemy-Guo-06        → 網師四部曲・少康第六集（高伯主控）
```

> 網師四部曲：網師主控的三部（回收／結網／刀與針）用 `Net`；少康部 POV 主角為高伯，用 `Guo`。

### 3.2 狀態欄位風格指南

狀態欄位以**簡練、帶張力的詞組**描述角色處境，避免平鋪直敘：

✅ 推薦：`身敗名裂 / 酗酒中`、`槍口下開刀 / 算計進行式`、`兩條線同時斷 / 止血中`
❌ 避免：`在醫院工作`、`執行任務中`、`準備行動`

## 四、正文結構

### 4.1 章節標題

以下列格式標示每個章節，不使用 Markdown 標題符號（Threads 不渲染 `#`）：

```
序章：{標題}
第一章：{標題}
第二章：{標題}
...
尾聲：{標題}
```

- **序章**：用於起源篇，提供時間軸錨點（「十五年前」）或世界觀背景
- **尾聲**：替代「結局」，帶有開放性或伏筆

### 4.2 段落節奏

- 每段以**短句收尾**，製造停頓感；長段之後接一句「呼吸句」
- 關鍵情緒翻轉以**獨立一行短句**強調，前後留白：

```
手術很成功。老人活了。
但李仁術「死」了。
```

- **對白**獨立成段，不與敘述混排
- 避免連續超過五行的純敘述，以對白或短句打散

### 4.3 強調格式

| 用途 | 格式 | 範例 |
|------|------|------|
| 角色代號、外號 | `**粗體**` | **「上帝之手」**、**「瘋狗」**、**「網師」** |
| 核心道具、組織名稱 | `**粗體**` | **【仁心堂】**、**C.H.I.A.N.G.** |
| 重要概念或反轉句 | `**粗體**` | **「從死神手裡搶人」**、**「敵方不可建模」** |
| 場所名稱（有象徵意義者） | `【方括號】` | 【仁心堂】、【異雲書屋】 |

> ⚠️ **原則**：粗體用於「如果拿掉，讀者會少感受到某個衝擊」的詞語。避免裝飾性濫用。

### 4.4 時間與空間標記

- 時間跳躍以**獨立行**標示，字體精簡：`十五年前。`、`三天後。`、`凌晨兩點。`
- 地點切換以**元資料地點箭頭**格式呼應，正文中視需要以場景描述帶出
- **反派本傳（非編年結構）特別注意**：部與部之間時間跨度大，每部開頭與必要的場景切換，須以獨立行給讀者明確時點路標（如 `一九九六年。`、`六年前。`、`現在。`），避免讀者迷失於倒敘。

## 五、結尾格式

### 5.1 待續標記

每集結尾統一以下列格式收尾，視類型選用：

```
【{系列標籤}（{集序}）．待續】
```

**範例：**
```
【李醫師前傳（一）．待續】
【Unleash Chiang（第3集）．待續】
【網師・回收（一）．待續】
```

### 5.2 大結局或完結篇

```
【{系列標籤}．完】
```
（反派本傳四部全完：`【網師四部曲・完】`）

### 5.3 番外篇（無集序）

```
【{篇名}．番外完】
```

## 六、Hashtag 規範

### 6.1 固定標籤（每篇必附，置於文末）

```
#行政院第九局 #九局宇宙 #台灣小說 #連載小說
```

### 6.2 類型標籤（依發文類型選用一至兩個）

```
#人物前傳      → Origin 類
#行動紀錄      → Mission 類
#九局密檔      → Classified 類
#動物番外      → 動物相關 Side 類
#UnleashChiang → War 類
#九局反派本傳   → Enemy 類（反派本傳，每篇必附；與系列標 #網師 並用）
```

### 6.3 角色標籤（依主角選用）

```
#李醫師 #阿豪大石 #林董福麥 #小潔修羅姬 #高伯分局長
#梅姐財務太后 #林姐白牡丹 #黃哥 #順仔Anderson
#威士忌士官長 #特工公主
#網師陸沉 #梭顧承 #餓狼 #謝明遠 #少康
```

**網師四部曲 Hashtag 範本：**
- 固定六標（每篇必附）：`#行政院第九局 #九局宇宙 #台灣小說 #連載小說 #九局反派本傳 #網師`
- 再依該集加一至兩個角色／主題標：`#網師陸沉`（全篇通用）／`#梭顧承`（回收）／`#少康 #高伯分局長`（少康）／`#餓狼 #謝明遠`（刀與針）

### 6.4 標籤擺放位置

- 所有 Hashtag **統一置於結尾標記之後**，以一個空行隔開
- 不在正文段落中插入 Hashtag，避免破壞敘事節奏

**範例結尾（反派本傳）：**
```
【網師・回收（一）．待續】

#行政院第九局 #九局宇宙 #台灣小說 #連載小說 #九局反派本傳 #網師 #網師陸沉
```

## 七、角色語言風格對照表

發文正文中若出現各角色的獨白或對白，應符合以下語言個性：

| 角色 | 語言特徵 | 範例 |
|------|----------|------|
| 阿豪 | 台語氣口、直白、有草根感 | 「來啊！老子什麼都沒有了！只有命一條！」 |
| 林董 | 表面軟爛、關鍵時刻精準狠辣 | 「這個嘛……（笑）你的軍火記錄我都有影本。」 |
| 李醫師 | 毒舌、簡短、專業詞彙與罵人混用 | 「沒有麻醉師又怎樣。你忍著點，大傢伙。」 |
| 小潔 | 高雅外殼、底下是極道氣場 | 「鞋跟刮了一道。要賠。」 |
| 高伯 | 留白、暗喻、每句都有三層意義 | 「重點不是這黑色的墨，而是留白。」 |
| 梅姐 | 會計體、鐵面、數字精確到小數點 | 「沒有統編。自己吞。」 |
| 林姐 | 茶道隱喻、優雅、不動聲色的威脅 | 「第一泡洗茶，把髒東西濾掉。」 |
| 順仔 | 初期：美式菁英腔；後期：台式同化 | 後期：「綠色乖乖，我懂了。」 |
| 威士忌 | 無對白，以行動與眼神敘述 | 牠閉上眼睛，把受傷的爪子伸了出來。 |
| 公主 | 無對白，以動作與神情敘述 | 公主用一只搖搖欲墜的盆栽陰影，制住了威士忌的A5和牛保溫箱。 |
| **網師** | **留白、暗喻、老茶人氣（與高伯同源，但更冷、更慢）；常用織造／棋局比喻** | **「我不抓魚。我織網，讓魚自己游進來。」** |
| **梭（顧承）** | **簡練、程序化、不露情緒，像在交報告；偶爾一句把荒謬說得很冷靜** | **「節點驗證通過。管道是活的。」** |

## 八、禁用事項

| 禁止行為 | 原因 |
|----------|------|
| 在正文中直接說明角色的心理狀態（Tell） | 違反「show, don't tell」原則 |
| 章節內使用 `#`、`##` 等 Markdown 標題符號 | Threads 不渲染，影響閱讀 |
| 跨集隨意更改元資料格式或檔案編號規則 | 破壞系列辨識度 |
| 在正文段落中間插入 Hashtag | 破壞敘事節奏 |
| 使用「突然」、「恰好」、「忽然」、「剛好」等巧合副詞推進關鍵劇情 | 違反角色動機邏輯 |
| 狀態欄填寫過於籠統的描述（如「任務中」） | 損失元資料的張力與資訊密度 |
| 使用破折號（——）製造停頓 | humanizer 風格：停頓改以句點＋換行處理 |

## 十、真實案件改編原則（反派本傳適用）

反派本傳若取材自真實、公開的歷史案件（如《少康》取材自少康專案／劉連昆案），依下列原則處理：

1. **可直接寫的歷史錨點**：採用公開、可考的史實事件（與正典既有的張憲義、尹清楓同一種用法），可作敘事錨點。
2. **虛構層標示**：九局與其虛構角色（高伯等）置入史實的「幕後策動」屬小說虛構設定，非史實主張。
3. **真實人物輕度虛構化**：凡會被賦予大量虛構對白與內心戲的真實人物，一律改用虛構姓名，並保留案件每一個可辨識的輪廓（讓認得此案的讀者一眼認出），但**不把虛構言行安到真實已歿者口中**。
4. **已公開言行作事實性引用**：真實、已公開的言行（如某次最高層的公開失言）可作事實性引用，不另行捏造其私下言行。
5. **基調**：涉及真實悲劇（如行刑）的章節，人的核心須莊重，不以喜劇處理。
