# 行政院第九局（Bureau 9）── 九局宇宙

> 台灣諜報喜劇連載小說 ｜ 荒謬的手段 × 嚴肅的目的
>
> [English version → README.en.md](README.en.md)

一個不存在於任何政府組織圖上的單位——行政院第九局。它的據點是鞋店、裝修行、茶行；它的特工是里長、廟會、電音三太子;它的武器,是這座島上人心裡真正在燒的東西。

本倉庫收錄「九局宇宙」全部已完稿作品,依系列分冊。所有作品皆以 Threads 連載格式寫成(每集 8 則貼文、每則 ≤500 字)。

## 主要系列

### 網師四部曲(反派本傳・已完稿)
以對岸情報線資深控管官「網師(陸沉)」為視角的反派本傳,與九局主線互為表裡。

- **第一部《回收》**(8 集 66 則)——一個結案的清晨,一張翻面的照片
- **第二部《少康》**(10 集 81 則)——1988–1999,一根深針的來歷
- **第三部《結網》**(8 集 64 則)——四敗,與一張永遠空白的臉
- **第四部《刀與針》**(10 集 80 則)——刀與針,先後斷在同一隻白手上;終格〈隔海一桌〉

### 春壤篇(外傳・已完稿)
謝明遠的思想養針本傳。一個一輩子在織網的人,發現一個自由的社會,是他唯一收不攏的網。17 集 136 則,三幕(織網/熄燈/收割不了),與《刀與針》交錯連載。

### 發文順序
《春壤篇》與《刀與針》採六區塊交錯連載,全域順序見 `07_設定集與準則/發文排程總表.md`。

## 創作準則(節錄)

- 每集 8 則、每則 ≤500 字;正文零破折號;禁用「突然/恰好」等巧合副詞
- Show, don't tell;角色行為符合動機邏輯;巧合不得推動關鍵劇情
- 台灣在地質地:台語氣口、里長/廟會/遶境等在地機構、用語經在地校準
- 鐵律:網師與白手,永不見面

完整準則見 `07_設定集與準則/`。
